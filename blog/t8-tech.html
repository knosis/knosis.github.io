This is for beginners

July 20, 2015

What is Rails? One thing that's always frustrated me is that most of the answers you need aren't in the dictionary or the encyclopedia. We hardly ever know why something is popular. We only know that it is. I always think back to the original inventor. Why did they create it? What issues were they having? How is the world different now.

It took me a while to find this answer. I had to watch about 100 ten minute videos, but I found it. If you are a beginner this might mean something to you otherwise it won't. But Rails is a program that was built to help you do all of the little annoying things you would need to do if you launched your own web application from scratch. How would you translate your database language into code your program could use? Would you remember to convert from HTML characters to URL characters? How often do little obstacles occur? Lots of things like that would require lots of work from lazy programmers. Ain't nobody got time for that.

It's built to help you launch quickly. That gives you more time to procrastinate.

Ruby Classes in Session

This is for beginners

June 20, 2015

Ruby classes solve a simple problem. Why create something from scratch every time when you can create a template and modify it? A class is very much like a template. It makes you think about which parts of the template will stay the same and which parts will be modified in the future. What is the minimum you'll need to get started?

Lets take this blog for example. There are certain parts of this blog which don't change much. If you look up top where the picture is, I'd say the header doesn't dance around from week to week. That footer is looking good just as it is. The most dynamic parts of this blog are the paragraphs of text and the links on the left which need to be updated. This is a pretty good example for creating a Ruby Blog class.
